# Miniguia de Estudos com NotebookLM

## 📌 Contexto e Objetivos
Este projeto foi desenvolvido como parte do desafio da DIO para explorar o uso da Inteligência Artificial como ferramenta de aprendizagem ativa.  
O objetivo é criar um **caderno temático no NotebookLM** que auxilie pessoas em **transição de carreira para análise de dados com tecnologia**, oferecendo um norte sobre quais trilhas seguir no ambiente atual de evolução das **IA's generativas**.

## 📖 Miniguia de Estudo (Entrega Final)

### Resumos Estruturados
- **Programação**:
A programação é a ferramenta que permite ao profissional manipular grandes volumes de informações e automatizar processos que seriam impossíveis de realizar manualmente.
  - **Lógica de Programação**: É o pensamento estruturado usado para resolver problemas de negócios e traduzir necessidades humanas em        instruções que o computador possa executar. Na análise de dados, a lógica é essencial para a **limpeza de registros**                   (identificar anomalias) e para a criação de fluxos de trabalho consistentes.
  - **Variáveis e Estruturas de Dados**: No contexto de dados, variáveis são usadas para armazenar valores temporários (como o                resultado de uma soma de vendas) que serão processados posteriormente por algoritmos.
  - **Estruturas de Repetição (Loops)**: São fundamentais para percorrer grandes conjuntos de dados (datasets), permitindo aplicar a          mesma regra de limpeza ou transformação em milhares de linhas de forma sequencial ou paralela.
  - **Linguagens Principais**: Python é a linguagem mais recomendada por sua versatilidade em automação, inteligência artificial e            análise estatística.
- **Banco de Dados**:
Bancos de dados são sistemas organizados para armazenar e recuperar informações corporativas, sendo a fonte primária de trabalho para analistas e engenheiros.
   - **SQL (Structured Query Language)**: É considerada a habilidade fundamental e obrigatória para qualquer carreira de dados. É a          linguagem usada para "conversar" com bancos de dados relacionais e extrair informações brutas.
   - **Tabelas**: São as estruturas básicas onde os dados ficam guardados em linhas (registros) e colunas (atributos), facilitando a         organização de **dados estruturados** como históricos de vendas ou cadastros de clientes.
   - **Consultas Simples (Queries)**: Referem-se ao ato de solicitar dados específicos ao servidor, permitindo filtrar apenas o que é        relevante (por exemplo, "listar apenas vendas do mês de maio") ou realizar agrupamentos para gerar estatísticas básicas.
- **IA Generativa**:
A IA Generativa representa um salto na forma como humanos interagem com máquinas, focando na criação de novos conteúdos a partir de modelos treinados.
    - **Modelos de Linguagem (LLMs)**: São o coração da IA generativa, como o GPT-4, Claude e Gemini. Eles são treinados em vastos            conjuntos de dados para entender, prever e gerar linguagem humana de forma contextualizada.
    - **Aplicações Práticas**:
        - **Engenharia de Prompt**: Otimização de instruções para obter respostas precisas da IA.
        - **Arquitetura RAG**: Técnica que conecta a IA a bases de dados privadas da empresa para fornecer respostas baseadas em fatos            reais, evitando erros de informação.
        - **Agentes Autônomos**: Sistemas que não apenas respondem perguntas, mas podem **executar ações**, como realizar buscas em               bancos de dados ou escrever código para resolver um problema.
    - **Impacto no Mercado**:
        - **Alta Demanda**: Empresas buscam profissionais (como o Engenheiro de IA) que saibam integrar essas ferramentas em softwares            existentes para melhorar a produtividade.
        - **Evolução do Trabalho**: A IA não substitui o analista de dados, mas serve como um assistente poderoso que acelera a                   descoberta de tendências e a geração de insights.
        - **Remuneração**: Profissionais especializados em IA e Machine Learning tendem a ter salários superiores devido à complexidade           técnica exigida.

### Glossário
- **RAG (Retrieval-Augmented Generation ou Geração Aumentada por Recuperação)**: uma forma de dar memória e contexto real a modelos de linguagem, evitando respostas genéricas ou inventadas.  
- **SQL (Structured Query Language ou Linguagem de Consulta Estruturada)**: linguagem padrão usada para interagir com bancos de dados relacionais, permitindo criar, consultar, atualizar e gerenciar dados de forma organizada.  
- **ML (Machine Learning ou Aprendizado de Máquina)**: área da IA focada em algoritmos capazes de aprender padrões em dados de treinamento e aplicar esse conhecimento para inferir resultados em novos dados.
