# Miniguia de Estudos com NotebookLM

## 📌 Contexto e Objetivos
Este projeto foi desenvolvido como parte do desafio da DIO para explorar o uso da Inteligência Artificial como ferramenta de aprendizagem ativa.  
O objetivo é criar um **caderno temático no NotebookLM** que auxilie pessoas em **transição de carreira para análise de dados com tecnologia**, oferecendo um norte sobre quais trilhas seguir no ambiente atual de evolução das **IA's generativas**.

## 🧠 Engenharia de Prompts e Cicatrizes
Durante o uso do NotebookLM, foram elaborados e testados diferentes prompts estratégicos:

### Exemplos de Prompts:
- "Quais são as trilhas mais comuns para iniciantes em tecnologia?"
- "Explique os conceitos básicos de SQL em linguagem simples."
- "Resuma os principais papéis em IA generativa."

### Cicatrizes (aprendizados e ajustes):
- Algumas respostas vieram muito genéricas → foi necessário refinar os prompts.  
- Testei variações como "Liste em tópicos" ou "Explique como se fosse para um iniciante".  
- Documentei as melhores respostas e os pontos de dificuldade para extrair informações mais úteis.

## 📖 Miniguia de Estudo (Entrega Final)

### Resumos Estruturados
- **Programação**: conceitos básicos de lógica, variáveis, estruturas de repetição.  
- **Banco de Dados**: SQL, tabelas, consultas simples.  
- **IA Generativa**: modelos de linguagem, aplicações práticas, impacto no mercado.  

### Glossário
- **Prompt**: instrução dada à IA para gerar uma resposta.  
- **Dataset**: conjunto de dados usado para treinar ou analisar.  
- **Modelo de Linguagem (LM)**: IA treinada para compreender e gerar texto.  
- **API**: interface que permite a comunicação entre sistemas.  

### Prompts Reutilizáveis
- "Resuma os principais conceitos de [tema] em até 5 tópicos."  
- "Explique [conceito] como se fosse para um iniciante."  
- "Liste exemplos práticos de aplicação de [tecnologia]."
